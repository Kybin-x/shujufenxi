💡 什么时候用哪个？
📊 用平均值，当：
• 数据比较均匀，没有极端值
• 想了解整体平均水平
• 如：班级平均成绩、日均气温
📊 用中位数，当：
• 数据有极端值（特别大或特别小）
• 想了解"大多数人"的水平
• 如：收入水平、房价分析
📊 用众数，当：
• 想知道"最受欢迎"是什么
• 想找最普遍的情况
• 如：卖得最多的尺码、最热门价格段

集中趋势统计量——数据的"代表值"

集中趋势 = 数据集中在哪个水平？就是"大家的整体表现怎样"。用于描述一组数据的"典型值"或"中心位置"。

📊
平均值（均值）
Mean / Average
=AVERAGE(B2:B21)
所有数据相加除以个数。最常用的集中趋势指标，但容易被极端数据影响。
举例：20天访客量均值 = 全部加起来 ÷ 20 = 126904 ÷ 20 = 6345.2
⚠️ 如果某天有爆款活动带来10923的UV，会把均值拉高
🎯
中位数
Median
=MEDIAN(B2:B21)
把所有数据从小到大排列后，最中间那个数。不受极端值影响，更真实。
举例：20个UV数据排序后，取第10、11位的平均值 = 6860
✅ 比均值6345更能代表大多数天的真实流量水平
🏆
众数
Mode
=MODE.SNGL(B2:B21)
出现次数最多的数。反映"最普遍"的情况，适合找最流行的值。
举例：20天UV数据中，6860出现了2次，是众数
📦 用途：找最热销的商品价格段、最常见的下单时间段
➕
总和
Sum
=SUM(B2:B21)
所有数据之和。用于计算总销售额、累计访客量等绝对量指标。
举例：20天总访客量 = 126904人次
📊 电商常用：月总销售额 = SUM(每天销售额)
🔍
解答开篇问题：为什么UV均值(6345) < 中位数(6860)？
活动初期（1-5日）访客量仅有 3200～4800，这些较低的数值把均值往下拖。
而双十一当天（11日）10923虽然很高，但只有1天，对中位数影响不大。
均值受低流量初期拖累 → 均值被拉低到6345
中位数不受极端值影响 → 稳定在6860
→ 中位数6860更能代表"大多数天"的真实流量！
📊 三个集中趋势指标对比
均值（AVERAGE） 6,345
中位数（MEDIAN） 6,860
众数（MODE） 6,860
离散程度统计量——数据的"波动幅度"

离散程度 = 数据波动有多大？差异有多悬殊？越大说明数据越不稳定，越分散。

📏
极差（全距）
Range
=MAX(B2:B21)-MIN(B2:B21)
最大值 − 最小值。反映数据的总体跨度，越大说明波动范围越广。
案例：10923 − 3127 = 7796
说明访客量最高和最低之间相差7796人次，波动范围非常大！
⚠️ 缺点：只看首尾两个数，中间数据情况不考虑
📉
标准差
Standard Deviation
=STDEV.S(B2:B21)
描述数据偏离平均值的程度。标准差越大，数据越分散、越不稳定，是最常用的离散程度指标。
案例：标准差 = 2231.38
说明每天访客量平均偏离均值约2231人，日流量波动非常明显
✅ 优点：考虑了所有数据，最全面
📐
方差
Variance
=VAR.S(B2:B21)
标准差的平方。和标准差一样用于衡量离散程度，但单位不同（是原单位的平方），不直观。
记住：方差 = 标准差²
2231.38² ≈ 4,979,070
实际工作中，用标准差更直观，方差用于统计计算
⬆️
最大值 / 最小值
Maximum / Minimum
=MAX() / =MIN()
数据的最高点和最低点。帮助快速发现异常情况和数据边界。
最大值：10923（双十一当天，活动效果极好）
最小值：3127（活动初期，预热不足）
两者相差3.5倍，说明活动影响力巨大
🎯
均值标准误差（S.E. Mean）
Standard Error of Mean
Excel描述统计自动输出
反映抽样误差的大小，越小说明平均值越可靠、越准确。用20天数据推断整个11月的均值时，这个误差有多大。
案例：S.E. Mean = 498.95
结合95%置信度：我们有95%的把握认为，11月整体日均UV在 6345 ± 1044 之间，即约 5301 ～ 7389 人/天
🎮

分布情况统计量——数据的"形态特征"

分布情况 = 数据是左多右少，还是均匀分布？用偏度和峰度来描述数据的形态特征。

📐 偏度（Skewness）——数据是否对称？
👈
左偏（负偏）
偏度 < 0
大多数数据集中在右边（较高值），少数极小值把"尾巴"拉向左边
电商例：大多数订单金额偏高，少数极低价清仓订单拉低左侧
⚖️
对称分布
偏度 ≈ 0
左右两边数据分布均匀，就是理想的"正态分布"（钟形曲线）
此时均值 = 中位数 = 众数，用均值最准确
👉
右偏（正偏）
偏度 > 0
大多数数据集中在左边（较低值），少数极大值把"尾巴"拉向右边
电商例：大多数UV较低，双十一爆发性高流量把右尾拉长
📈 峰度（Kurtosis）——数据集中还是分散？
🏔️
尖顶峰（高峰）
峰度 > 0
数据高度集中在均值附近，极端值少，分布曲线比正态分布更"尖"更细
大多数天UV都集中在某个区间，偶尔的高低波动很少
🔔
正态（标准）
峰度 ≈ 0
标准钟形曲线，是最理想状态。许多统计方法都基于正态分布的假设
判断方法：偏度在±1之间 + 峰度在±2之间 → 基本可认为正态
🏝️
平顶峰（低峰）
峰度 < 0
数据较分散，分布比较均匀，曲线顶部比正态分布更"平坦宽阔"
UV数据峰度=-0.70，略平顶，说明数据分布比较均匀分散
💡 记忆口诀
偏度峰度都接近0，就是正态分布！
实际工作中，偏度在 ±1 之间、峰度在 ±2 之间，基本可认为接近正态分布。
案例验证： 访客UV数据 偏度=0.23，峰度=-0.70 → 接近正态 ✅
✅ 接近正态分布时，均值、中位数、众数三者很接近，用均值分析最合适。偏离正态越多，越要优先考虑中位数。

练习一 · 判断题
每题 +8 XP

根据所学知识判断以下说法是否正确：

1. 当数据中存在极端值（如超高工资的老板）时，用中位数比均值更能代表大多数人的典型水平。
2. 偏度的绝对值越大，说明数据越接近对称分布，越接近正态分布。
3. 两组数据均值相同，但A组标准差=200，B组标准差=3800，说明A组数据更稳定可预测。
4. Excel的"数据分析"工具库在安装Office时已经默认加载，无需手动操作即可使用。
5. 描述统计中的均值标准误差（S.E. Mean）越小，说明用样本均值推断总体均值的可靠性越高。

📌 本单元核心要点
1
三类统计量：集中趋势（均值/中位数/众数/总和）、离散程度（标准差/方差/极差/最大最小值）、分布情况（偏度/峰度）
2
指标选择原则：有极端值→中位数；无极端值→均值；找最流行→众数；评稳定性→标准差
3
标准差口诀：标准差小=稳定；标准差大=波动大。均值相同时，标准差才是区分稳定性的关键
4
判断正态分布：偏度在±1内且峰度在±2内，基本可认为近似正态分布
5
Excel操作路径：文件→选项→加载项→勾选分析工具库 → 数据→数据分析→描述统计→勾选汇总统计
6
中位数>均值说明有极低值拉低了均值；中位数<均值说明有极高值拉高了均值
