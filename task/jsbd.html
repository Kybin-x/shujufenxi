📋 任务背景

🔗 承接任务一：二手数据 → 一手数据

巨量算数只能告诉你"看了什么、搜了什么"，但消费者为什么买、最在意什么，得直接去问

完成任务一的巨量算数数据分析后，选品经理小李说："指数数据帮我们搞清楚了哪些品类在抖音上最热、旺季在什么时候。但我们还想知道——消费者买玩具最看重什么？在哪个平台买？愿意花多少钱？这些问题，得用问卷直接去问。"

于是你被安排设计一份消费者购买行为调查问卷，通过金山表单发布，向身边的家长群体收集第一手数据，作为抖音指数数据的重要补充，让选品决策更有说服力。
💡 核心任务：用金山表单设计一份包含9道题的调查问卷，发布并收集不少于10份有效回答，最后提炼3条数据结论——并与任务一的抖音指数数据进行对比印证。

📚 操作前必读：问卷设计要点
✅ 好问卷的特征
  • 题目数量适中（8-12题），完成时间不超过5分钟
  • 每道题只问一件事，不写"又A又B"的复合问题
  • 选项互斥穷举（不重叠、不遗漏），必要时加"其他"
  • 敏感问题（如收入）放后面，先建立信任感
  • 开头说明调查目的，承诺数据只用于学习研究
❌ 常见设计错误
  • 诱导性问题：如"您是否认为高价玩具更好？"
  • 模糊选项："偶尔""经常"没给出具体频次说明
  • 题目太多：超过15题导致大量中途放弃
  • 全部必填：部分开放题设为非必填更真实
  • 双重否定句：让被调查者难以理解真实意图
🗂️ 问卷结构规划（共9题）
以下是本次实训的完整问卷题目设计方案，请严格按照此结构在金山表单中创建（可微调题目文字，但题型和结构不变）。
1
您的孩子目前的年龄段是？单选题
选项：0-2岁 / 3-5岁 / 6-8岁 / 9-12岁 / 暂无孩子（此题为过滤题，用于了解受访者群体构成）
2
您家孩子目前拥有的益智玩具类型？（可多选）多选题
选项：积木/拼装类 / 拼图类 / 桌游/棋类 / 科学实验类 / 绘画/手工类 / 其他（请补充）
3
您通常在哪些渠道购买儿童益智玩具？（可多选）多选题
选项：淘宝/天猫 / 京东 / 拼多多 / 抖音小店/直播间 / 线下实体店 / 其他（请补充）
4
您每次购买儿童益智玩具的单次消费金额通常是？单选题
选项：50元以下 / 50-100元 / 100-200元 / 200-500元 / 500元以上
5
购买益智玩具时，您最看重哪些因素？（最多选3项）多选题
选项：安全性/材质 / 益智功能/教育性 / 品牌知名度 / 价格 / 外观/颜色 / 年龄适配 / 买家好评数量 / 主播/达人推荐
6
您通常在什么时候给孩子购买玩具？（可多选）多选题
选项：生日/节日 / 孩子主动要求时 / 日常随机购买 / 促销活动期间 / 刷到短视频/直播后冲动购买
7
您通过哪些途径了解到新的儿童玩具？多选题
选项：抖音短视频/直播种草 / 小红书图文 / 亲友推荐 / 电商平台推荐 / 育儿公众号/社群 / 线下体验
8
您对目前市场上儿童益智玩具的满意程度？评分题（1-5星）
1=非常不满意，5=非常满意。评分题用于直观呈现整体满意度，后台会自动计算均值。
9
您认为目前儿童益智玩具市场最大的不足是什么？填空题（非必填）
开放式回答，收集消费者真实痛点，为选品改进和内容创作提供方向参考。
🔧 操作步骤（共6步）
1
登录金山文档，新建表单并完成基础设置
找到金山表单功能入口，写好开场白
▼
  • ①
    打开浏览器访问 www.kdocs.cn，用微信/手机号登录金山文档账号
  • ②
    点击左上角"新建"→ 选择"表单"（注意：不是表格），进入表单编辑界面
  • ③
    点击顶部标题区域，改为：儿童益智玩具购买行为调查
  • ④
    在标题下方的描述文字框填写开场白："您好！我是电商数据分析专业的学生，本次调查旨在了解家长对儿童益智玩具的购买习惯和偏好，调查结果仅用于学习研究，大约需要3分钟，感谢配合！"
💡
好的开场白能提高填写率，关键要包含：① 调查目的；② 所需时间；③ 数据保密承诺。
2
按规划逐题创建9道题目
使用单选、多选、评分、填空四种题型
▼
📌
金山表单共4种题型用于本次问卷，操作前先了解各自用途：
单选题只能选一个选项
适用：互斥选项（年龄段、消费金额）
操作：点击"+" → 选择"单选题"
多选题可选多个选项
适用：购买渠道、考虑因素等并列内容
操作：点击"+" → 选择"多选题"
评分题1-5星打分
适用：满意度评价
操作：点击"+" → 选择"评分"
填空题开放式文字回答
适用：意见建议等开放回答
操作：点击"+" → 选择"填写框"
  • ①
    按照上方"问卷结构规划"的9道题，逐题添加（注意选择正确的题型）
  • ②
    第1-8题设为必填，第9题（填空）设为非必填——强制填写所有题会降低完成率
  • ③
    每道多选题的选项末尾加上"其他（请补充）"，避免遗漏特殊情况
  • ④
    全部题目添加完后，点击右上角"预览"，以填写者视角检查流畅度和完整性
⚠️
预览时重点检查：题目表述是否清晰？选项有无遗漏常见情况？有没有诱导性或模糊性措辞？
3
配置表单设置，写好感谢结束语
设置截止时间、答题限制和感谢页内容
▼
  • ①
    点击顶部"设置"图标，进入表单配置页面
  • ②
    截止时间：设置为3天后（给被调查者足够时间填写）
  • ③
    每人只能提交一次：建议开启，防止重复提交影响数据质量（开启后填写者需要登录）
  • ④
    找到"结束语/感谢页"设置，填写："感谢您完成本次调查！您的回答将帮助我们更准确地了解消费者需求，非常感谢您的支持！"
📸
截图要求1：截取金山表单的完整编辑界面（能看到全部9道题的列表），保存为"截图1_问卷设计.png"
4
发布问卷，收集10份以上回答
获取分享链接，通过微信/班级群分发
▼
  • ①
    点击右上角"发布"或"收集回答"按钮，获取分享链接和二维码
  • ②
    将链接/二维码通过微信发给家人、朋友，或发布到班级群请同学帮忙填写
  • ③
    发送时附上简短说明："同学你好，这是我电商数据分析课的实训作业，一份关于儿童玩具的调查问卷，大约3分钟，谢谢帮忙！[链接]"
  • ④
    目标是收集不少于10份有效回答（如果班级互填，每人都有机会达到）
💡
发问卷时附上简短说明（目的+所需时间）能显著提高回复率。如果身边被调查者不多，可以和同班同学互相填写对方的问卷。
5
查看统计后台，读取自动生成的图表数据
进入金山表单数据统计页面，逐题分析结果
▼
  • ①
    收到足够回答后，在金山文档中打开该表单，点击"查看结果"或"数据统计"
  • ②
    查看整体概况：总回答数、平均完成时长、提交时间分布
  • ③
    逐题查看自动统计图表（单选/多选题会自动生成饼图或柱状图），记录每道题占比最高的选项
  • ④
    查看第9题填空的文字回答列表，找出被提及最多的关键词和痛点
  • ⑤
    如需深入分析，可点击"导出数据"将原始数据下载为 Excel 格式
📸
截图要求2：截取统计后台的整体数据概览页面（包含回答总数和各题统计图），保存为"截图2_数据统计.png"
6
撰写分析小结，与抖音指数数据对比验证
提炼结论，联系任务一的巨量算数分析结果
▼
  • ①
    围绕"谁在买 → 在哪买 → 怎么发现的 → 买什么 → 看重什么"五个维度整理问卷数据
  • ②
    对照任务一的抖音指数结论，看看两者是否相互印证。例如：巨量算数显示六一前综合指数飙升，问卷中是否有大量人回答"节日前购买"？
  • ③
    写出至少3条数据驱动的结论，格式参考："根据问卷数据，X%的受访者通过抖音短视频/直播了解到玩具，建议优先制作短视频内容而非图文。"
✅
好结论的格式：数据来源 → 发现的规律 → 对选品/运营的建议。不要只说"大部分人选了X"，要说"X%选择了X，建议……"，有数字才有说服力。
📊 重点关注哪些题的数据？
🛒
购买渠道（第3题）
抖音小店/直播间占比多少？这直接验证了任务一中抖音热度数据的"变现可行性"。如果抖音渠道占比高，说明这个品类在抖音有真实成交，值得布局。
📱
信息获取渠道（第7题）
消费者主要通过抖音还是小红书被种草？这决定内容营销的主战场。结合任务一的综合指数（内容量）来对比：指数高的平台是否也是消费者的主要发现渠道？
💰
消费金额（第4题）
主流价格带在哪里？这直接指导选品的客单价定位。如果问卷显示主流是100-200元，选品时要卡准这个区间，避免选品价格与市场预期脱节。
⭐
核心购买考量（第5题）
消费者最看重什么？这决定短视频内容的核心卖点。如果安全性排第一，视频脚本就应突出材质/认证信息；如果"达人推荐"占比高，说明种草内容的信任背书很重要。
