3.3.2 格式内容清洗

统一日期格式 · 去除多余空格 · 规范金额与商品名称 · 处理错误值
📁 配套文件：3.3.2_练习数据.xlsx 🎯 三个难度等级 🏢 情景：星途电商有限公司

📋 背景情景

星途电商有限公司每月从各平台导出销售订单数据， 由于不同平台的导出格式不同、不同操作人员录入习惯各异， 数据中常常出现日期格式混乱、商品名称前后有空格、金额带单位符号等问题， 导致函数计算出错、排序不准确。 数据助理小何需要在正式分析前完成数据格式的全面清洗。 请协助小何完成各难度的格式清洗任务。

📖 知识点速览：常见格式问题与清洗方法

📅 日期格式不统一
❌ 2024/3/15 / 20240315 / 2024年3月15日
✓ 2024-03-15
方法：选中列→设置单元格格式→日期→选择标准格式
🔲 多余空格
❌ " 快充数据线 " / "手 机 壳"
✓ "快充数据线" / "手机壳"
方法：=TRIM(A2) 去除首尾和多余中间空格
💰 金额格式问题
❌ "¥89.00" / "89元" / "89,00"
✓ 89.00（纯数值）
方法：查找替换去除符号，或用VALUE()转换
🔤 大小写/全半角
❌ "type-c" / "TYPE-C" / "Ｔｙｐｅ－Ｃ"（全角）
✓ "Type-C"
方法：PROPER()/UPPER()/LOWER() 或查找替换
⚠️ 错误值 #DIV/0!
❌ =B2/C2 → #DIV/0!（除以0报错）
✓ =IFERROR(B2/C2, "--")
方法：用IFERROR()包裹原公式，指定出错时的替代值
📝 省份/类别名称不统一
❌ "广东省" vs "广东" / "手机保护壳" vs "手机壳"
✓ 统一为标准名称
方法：查找替换（Ctrl+H）批量修改
⚠️ 清洗原则：不要直接修改原始数据列， 建议在旁边新增"清洗后"列，用函数处理后再复制→选择性粘贴（仅值）覆盖， 或保留原始列备份。

✅ 任务要求
  1. 将数据粘贴到"数据（基础）"Sheet，不要修改原始列，在 F 列新建"商品名称（清洗后）"，在 G 列新建"下单日期（清洗后）"。
  2. 清洗商品名称：在 F2 输入公式 =TRIM(B2)， 向下填充到 F26，去除所有首尾及多余空格。
  3. 清洗日期格式：
    • 对于"2024/3/15"格式：选中该列 → 设置单元格格式 → 日期 → 选择"2012-03-14"样式
    • 对于"2024年3月15日"格式：查找替换（Ctrl+H），将"年"替换为"-"，"月"替换为"-"，"日"替换为""
    • 对于"20240315"格式（8位数字）：需用 TEXT+MID 函数组合转换，在 G 列写公式
  4. 完成清洗后，验证结果：G 列所有日期应能正确参与排序（按升序排列验证）。
  5. 将清洗操作记录填入下方操作记录表。
🔧 日期清洗操作路径
  1. 格式统一（斜线日期）：选中 D 列 → 右键 → 设置单元格格式 → 数字 → 日期 → 选择"2012-03-14"→ 确定
  2. 中文日期转换：Ctrl+H → 查找"年"→ 替换为"-"→ 全部替换； 再次查找"月"→ 替换为"-"→ 全部替换； 再次查找"日"→ 替换为""→ 全部替换
  3. 纯数字日期（如20240315）转换： 在 G 列辅助列输入： =TEXT(D2,"0000-00-00")（若为数值型） 或 =LEFT(D2,4)&"-"&MID(D2,5,2)&"-"&RIGHT(D2,2)（若为文本型）
  4. 全部处理完成后，选中 G 列 → 复制 → 选择性粘贴（仅值）→ 粘贴到 D 列

💡 操作提示
  • TRIM 函数：只去除首尾空格和单词间多余空格（保留单词间一个空格），不能去除字符串中间所有空格。 如需去除所有空格，用 =SUBSTITUTE(B2," ","")。
  • 判断日期是否真正为日期格式：选中单元格，若左下角显示的是日期（如"2024-3-15"）， 说明 Excel 识别为日期；若显示为数字（如45366），也是日期（Excel内部日期存储为数字）； 若显示为文本，则需要转换。
  • 选择性粘贴（仅值）：复制公式结果 → 在目标单元格右键 → 选择性粘贴 → 勾选"值"→ 确定。这样可以把公式结果转为静态数值。
  • 如果部分日期处理完后仍显示为文本（左对齐）， 可选中该列 → 数据 → 分列 → 直接点完成，强制让 Excel 重新识别格式。

✅ 任务要求
  1. 将数据粘贴到"数据（巩固）"Sheet，先统计各字段的格式问题数量，填入问题统计表。
  2. 清洗商品名称：
    • 去除首尾空格：=TRIM(B2)
    • 统一 Type-C 大小写：使用查找替换，将"type-c""TYPE-C"统一替换为"Type-C"
  3. 清洗省份名称：使用查找替换（Ctrl+H）， 将"广东省"替换为"广东"，"浙江省"替换为"浙江"，依此类推。
  4. 清洗销售额：
    • 去除"¥"符号：查找替换，将"¥"替换为""（空）
    • 去除"元"字：查找替换，将"元"替换为""（空）
    • 转换为数值：用 =VALUE(D2) 函数，或数据→分列→完成
  5. 清洗下单日期（方法同基础练习，参考操作路径提示）。
  6. 验证清洗结果：
    • 对销售额列用 SUM 函数求和，若结果正确（非0）说明已转为数值
    • 对日期列按升序排序，验证顺序是否正确
    • 对省份列用 COUNTIF 统计"广东"的数量，确认无"广东省"残留
  7. 完成所有清洗后，填写清洗结果验证表。

💡 操作提示
  • 销售额带"¥"符号后无法求和：因为带符号的数字被存储为文本。 查找替换去除符号后，如果 SUM 仍为 0，还需要用 =VALUE() 转换， 或选中列→数据→分列→完成，强制转为数值型。
  • 查找替换的注意事项：查找"元"时，如果商品名称中含有"元"字（如"万元充电宝"）， 会被一起替换。建议勾选"单元格匹配"选项，或只选中销售额列再执行替换。
  • 省份统一的快捷方式：如果省份种类较多，可以用 =SUBSTITUTE(C2,"省","") 函数一次性去除所有"省"字， 比逐个查找替换更高效。
  • 验证时发现未通过：重新检查该字段是否有遗漏的格式问题， 用"查找"功能（Ctrl+F）搜索残留的符号或文字。

✅ 任务要求
  1. 第一步：清洗推广渠道列。 用 =SUBSTITUTE(B2," ","") 去除所有空格 （注意：渠道名称中间的空格也要去除，用SUBSTITUTE而非TRIM）。
  2. 第二步：规范商品类别名称。 将"充电宝类""充电宝-新"等非标准名称统一为"充电宝"， 其他类别同理，使用查找替换完成。
  3. 第三步：清洗金额列。 去除推广花费和带来销售额列中的"元""￥"符号，并用 =VALUE() 转换为数值型。
  4. 第四步：处理转化率列的 #DIV/0! 错误。 找到转化率列的原始公式（形如 =成交数/点击量）， 用 IFERROR 包裹，使出错时显示 "--"：

  5. 清洗记录日期列（方法同前两个难度）。
  6. 第五步：全面验证。 使用下方验证清单，逐项检查清洗结果，确保所有字段达标。
  7. 第六步：撰写清洗日志。 在新 Sheet（命名"清洗日志"）中，按下方模板记录完整的清洗过程。

💡 操作提示
  • TRIM vs SUBSTITUTE 去空格的区别：
    TRIM：去除首尾空格 + 单词间多余空格（保留一个）
    SUBSTITUTE(B2," ","")：去除所有空格（包括中间的）
    渠道名如"直 通 车"中间有空格，用 SUBSTITUTE 更合适。
  • 查找类别非标准名称的方法：数据→数据透视表，把商品类别放到行区域， 可快速看出有哪些非标准名称（如出现"充电宝类""充电宝-新"等）。
  • IFERROR 处理转化率：转化率原公式可能形如 =G2/F2（成交数/点击量）， 改写为 =IFERROR(G2/F2,"--")。 注意：如果该列已经显示 #DIV/0!，说明公式已存在但点击量为0， 直接编辑该单元格公式，在外层加 IFERROR 即可。
  • 清洗日志的价值：清洗日志是数据处理的"操作记录"， 便于他人复查和后续维护。填写时要具体到每个操作步骤， 不能只写"已清洗"这类模糊表述。
📌 进阶拓展：一次性检查所有格式问题的思路
  • 对文本字段：用 =LENB(A2)-LEN(A2) 检测是否含全角字符 （全角字符LENB比LEN多1）。
  • 对数值字段：用 =ISNUMBER(D2) 检测是否为真正数值， 结果为 FALSE 说明该单元格是文本型数字。
  • 对日期字段：用 =ISNUMBER(E2) 检测， Excel日期本质是数值，若为 FALSE 说明是文本型日期，需要转换。
  • 批量检测完成后，筛选出 FALSE 的行，集中处理。
