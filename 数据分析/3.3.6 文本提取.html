3.3.6 LEFT / MID / RIGHT 文本提取

从固定格式文本中精准提取所需片段，配合 FIND/LEN 处理变长字段
📁 配套文件：3.3.6_练习数据.xlsx 🎯 三个难度等级 🏢 情景：星途电商有限公司

📋 背景情景

星途电商有限公司客服部门导出了一批客户信息和商品 SKU 编码数据。 这些数据中，姓名、手机号、城市代码等信息被混在一个字段里， 商品 SKU 也是用"-"分隔的多段编码。 数据分析实习生小方需要将这些"合并字段"拆分提取， 分别存入独立的列中，便于后续统计分析。 请协助小方使用 LEFT、MID、RIGHT 及配合 FIND、LEN 函数完成文本提取任务。

📖 知识点速览：三个文本提取函数

LEFT
=LEFT(文本, 字符数)
  • 文本：要提取的原始文本
  • 字符数：从左边取几个字符
示例：
=LEFT("张三李四", 2)
→ "张三"（取左边2个字符）
MID
=MID(文本, 起始位置, 字符数)
  • 文本：要提取的原始文本
  • 起始位置：从第几个字符开始
  • 字符数：取几个字符
示例：
=MID("DXL-BK-1M", 5, 2)
→ "BK"（从第5位取2个字符）
RIGHT
=RIGHT(文本, 字符数)
  • 文本：要提取的原始文本
  • 字符数：从右边取几个字符
示例：
=RIGHT("北京0001", 4)
→ "0001"（取右边4个字符）
📍 字符位置示意（以 "王小明13800138001广东0001" 为例）：
王 小 明 1 3 8 0 0 1 3 8 0 0 1 广 东 0 0 0 1
LEFT(A2,3) = "王小明"（姓名3字）
MID(A2,4,11) = "13800138001"（手机11位）
RIGHT(A2,6) = "广东0001"（城市代码6位）
⚠️ 注意：LEFT/MID/RIGHT 中的"字符数"以字符个数计算， 汉字和英文字母、数字都算1个字符。 如果需要按字节计算（汉字=2字节），要用 LEFTB/MIDB/RIGHTB， 但本课程统一使用字符版本（LEFT/MID/RIGHT）。

基础练习

适合：第一次使用文本提取函数 · 25条客户信息 · 格式固定，三段信息
📊 数据说明：25条客户信息｜格式固定： 3字姓名 + 11位手机号 + 城市名(2字) + 4位客户编号｜ 用 LEFT/MID/RIGHT 分别提取三段

字段名 列 说明 格式说明

消费金额（元） B列 该客户的累计消费金额 数值型，无需提取
会员等级 C列 普通/银牌/金牌/钻石 文本，无需提取

张 三 丰 13 80 01 38 00 1 广 东 0 0 0 1

✅ 任务要求
  1. 将数据粘贴到"数据（基础）"Sheet，在 D、E、F、G 列分别新建： "姓名""手机号""所在城市""客户编号"四列。
  2. 在 D2 输入提取姓名的公式，向下填充到 D26：
    =LEFT(A2, 3)
  3. 在 E2 输入提取手机号的公式，向下填充到 E26：
    =MID(A2, 4, 11)
  4. 在 F2 输入提取城市名的公式，向下填充到 F26：
    =MID(A2, 15, 2)
  5. 在 G2 输入提取客户编号的公式，向下填充到 G26：
    =RIGHT(A2, 4)
  6. 完成后验证结果，并完成下方统计练习。

📝 提取结果统计（在 Excel 中用函数完成）
统计项目 使用函数 公式示例 结果
来自"广东"的客户数量 COUNTIF =COUNTIF(F2:F26,"广东")
广东客户的平均消费金额 AVERAGEIF =AVERAGEIF(F2:F26,"广东",B2:B26)
金牌会员中广东客户数 COUNTIFS =COUNTIFS(F2:F26,"广东",C2:C26,"金牌")
手机号以"138"开头的客户数 COUNTIF =COUNTIF(E2:E26,"138*")
💡 操作提示
  • 字符位置从1开始计数： MID(A2, 4, 11) 表示从第4个字符开始，取11个字符。 第1个字符是"张"，第4个字符是"1"（手机号第一位）。
  • 验证提取是否正确：提取完成后， 用 =LEN(E2) 检查手机号长度是否为11， 用 =LEN(D2) 检查姓名是否为3。 若长度不对，说明提取参数有误。
  • 为什么城市名用MID而不是RIGHT： 城市名在第15-16位，后面还有4位编号， 用RIGHT(A2,6)会得到"广东0001"（6位）， 只需城市名则用MID(A2,15,2)更精确。
  • 提取出的手机号是文本型数字（左对齐）， 这是正常的，手机号本来就应该存为文本（不参与数值运算）。
LEVEL 2

巩固练习

适合：熟悉基本函数后 · 28条SKU数据 · 拆分编码+按类别统计
📊 数据说明：28条商品SKU数据｜ SKU格式：类别代码(3位)-颜色代码(2位)-尺寸代码(2位)｜ 用 MID 提取颜色代码，用 LEFT/RIGHT 提取类别和尺寸
📌 SKU编码格式说明
🏷️ SKU编码结构（格式固定：类别-颜色-尺寸）
DXL - BK - 1M
说明：
  • 类别代码（3位）：DXL=数据线，CPB=充电宝，PHC=手机壳，BTE=蓝牙耳机，MGC=磁吸充电器
  • 颜色代码（2位）：BK=黑色，WH=白色，RD=红色，BL=蓝色，GN=绿色
  • 尺寸代码（2位）：1M=1米，2M=2米，5K=5000mAh，10K=10000mAh，IN=入耳，OV=头戴
提取公式：
类别代码：=LEFT(A2, 3) → "DXL"
颜色代码：=MID(A2, 5, 2) → "BK"（跳过第4位的"-"，从第5位取2位）
尺寸代码：=RIGHT(A2, 2) → "1M"
📌 数据字段说明
字段名 列 说明
SKU编码 A列 格式固定：类别(3)-颜色(2)-尺寸(2)，共8位（含两个"-"共10位）
库存数量（件） B列 当前库存
单价（元） C列 商品售价
近30天销量（件） D列 近30天出售数量
✅ 任务要求
  1. 将数据粘贴到"数据（巩固）"Sheet，在 E、F、G 列分别新建： "类别代码""颜色代码""尺寸代码"三列，用公式提取。
  2. 在 H 列新建"商品类别（中文）"列， 用 IF 嵌套将类别代码转为中文名称：
    =IF(E2="DXL","数据线",IF(E2="CPB","充电宝",IF(E2="PHC","手机壳",IF(E2="BTE","蓝牙耳机","磁吸充电器"))))
  3. 在 I 列新建"颜色（中文）"列，同理将颜色代码转为中文（黑/白/红/蓝/绿）。
  4. 完成提取后，在 Sheet 右侧建立"汇总统计区"， 用 SUMIF/COUNTIF 完成下方统计表。
  5. 分析各类别的库存和销售情况，填写分析结论。

💡 操作提示
  • MID提取颜色代码的位置计算： SKU格式为 DXL-BK-1M，字符位置如下： D(1) X(2) L(3) -(4) B(5) K(6) -(7) 1(8) M(9) 颜色代码从第5位开始，取2位，所以是 =MID(A2,5,2)。
  • IF嵌套转中文名称： 5种类别需要嵌套4层IF，逻辑为： IF(是DXL, 数据线, IF(是CPB, 充电宝, IF(是PHC, 手机壳, IF(是BTE, 蓝牙耳机, 磁吸充电器)))) 最后一个不需要再判断，直接写默认值即可。
  • SUMIF按类别汇总： 条件区域是 H 列（类别中文名），条件是"数据线"等， 求和区域分别是 B 列（库存）或 D 列（销量）。
  • 库存消化率= 近30天销量 ÷ 当前库存， 反映库存周转速度，值越高说明销售越快、库存压力越小。

进阶练习

适合：综合运用阶段 · 30条数据 · 变长字段需配合 FIND/LEN 动态提取
📊 数据说明：30条混合数据｜ 商品名称长度不固定，需配合 FIND/LEN 动态定位提取位置｜ 包含多种格式的合并字段
📐 进阶工具：FIND 和 LEN 函数
函数 语法 用途 示例
FIND =FIND(查找文本, 字符串, [起始位置]) 返回某字符在字符串中的位置 =FIND("-","DXL-BK-1M") → 4（第一个"-"在第4位）
LEN =LEN(文本) 返回字符串的总字符数 =LEN("快充数据线") → 5（共5个字符）
组合用法：动态提取"-"之前的内容
=LEFT(A2, FIND("-",A2)-1) → 提取第一个"-"之前的所有字符（不管前面有几个字符）

动态提取"-"之后的内容
=MID(A2, FIND("-",A2)+1, LEN(A2)-FIND("-",A2)) → 提取第一个"-"之后的所有字符
📌 数据字段说明
字段名 列 格式描述 提取难点
商品规格描述 A列 格式：商品名称-规格参数
例："快充数据线Type-C-1M" / "充电宝-10000mAh"		 商品名称长度不固定，用FIND定位"-"位置
供应商_城市 B列 格式：供应商名称_城市
例："深圳华联科技_深圳" / "东莞聚能电子_东莞"		 供应商名称长度不固定，用FIND定位"_"位置
入库批次 C列 格式：年份(4位)-月份(2位)-批次号(3位)
例："2024-03-001"		 格式固定，用LEFT/MID/RIGHT直接提取
入库数量（件） D列 数值型 无需提取
✅ 任务要求
  1. 将数据粘贴到"数据（进阶）"Sheet，在右侧建立提取列： E列"商品名称"、F列"规格参数"、G列"供应商名称"、H列"城市"、I列"年份"、J列"月份"、K列"批次号"。
  2. 提取商品名称（A列"-"之前的内容，长度不固定）：
    =LEFT(A2, FIND("-",A2)-1)
    注意：部分商品名称中含有多个"-"（如"Type-C"）， 需要用 FIND("-",A2,FIND("-",A2)+1) 定位第二个"-"。
  3. 提取规格参数（A列最后一个"-"之后的内容）：
    使用 LEN 和 FIND 组合动态计算：
    =RIGHT(A2, LEN(A2)-FIND("☆",SUBSTITUTE(A2,"-","☆",LEN(A2)-LEN(SUBSTITUTE(A2,"-","")))))
    （此为提取最后一段"-"后内容的进阶公式，操作提示中有简化方法）
  4. 提取供应商名称和城市（B列"_"分隔）：
    • 供应商名称：=LEFT(B2, FIND("_",B2)-1)
    • 城市：=RIGHT(B2, LEN(B2)-FIND("_",B2))
  5. 提取入库批次各段（C列格式固定：年-月-批次号）：
    • 年份：=LEFT(C2, 4)
    • 月份：=MID(C2, 6, 2)
    • 批次号：=RIGHT(C2, 3)
  6. 完成提取后，用 COUNTIF 统计各城市供应商数量，填入下方统计表。
✏️ 关键公式填写（变长字段部分）
1 提取商品名称（"-"之前，长度不固定）
示例：充电宝-10000mAh（"充电宝"3个字，"-"在第4位）
= LEFT( A2, FIND("-", A2) - _____ )
预期结果：充电宝（不含"-"）
2 提取供应商名称（"_"之前，长度不固定）
示例：深圳华联科技_深圳（"_"在第7位）
= LEFT( B2, FIND("_", B2) - _____ )
预期结果：深圳华联科技
3 提取城市（"_"之后，用LEN-FIND动态计算）
示例：深圳华联科技_深圳（总长度10，"_"在第7位，城市从第8位到最后）
= RIGHT( B2, LEN(B2) - FIND("_", B2) )
预期结果：深圳（RIGHT取的字符数 = 总长度 - "_"的位置）
4 提取批次号月份（固定格式：2024-03-001，月份在第6-7位）
示例：2024-03-001
= MID( C2, _____ , _____ )
预期结果：03

  2. 先用分列功能：选中A列 → 数据 → 分列 → 分隔符号 → 勾选"-" → 完成。 Excel会自动将内容按"-"拆分为多列。

💡 操作提示
  • FIND函数找不到字符时会报错： 若数据中某行没有"-"或"_"，FIND会返回#VALUE!错误。 可用 =IFERROR(LEFT(A2,FIND("-",A2)-1), A2) 保护， 出错时直接显示原始文本。
  • 提取最后一段的LEN-FIND方法解析： =RIGHT(B2, LEN(B2)-FIND("_",B2))
    LEN(B2) = 总字符数，FIND("_",B2) = "_"所在位置， 两者相减 = "_"之后的字符数，RIGHT取这么多字符就是"_"之后的内容。
  • 商品名称含多个"-"的处理： "快充数据线Type-C-1M"中有两个"-"，第一个"-"在"Type"和"C"之间， 不是我们要找的分隔符。 可改用分列功能（数据→分列→分隔符"-"）， Excel会拆成三列，取最后一列为规格参数。
  • 月份提取结果为文本"03"不是数字3： 这是正常的，用COUNTIF/SUMIF时条件写"03"（加引号）即可匹配。
📌 LEN + FIND 组合公式的逻辑总结
  • 提取分隔符之前：=LEFT(A2, FIND("分隔符",A2)-1)
  • 提取分隔符之后：=RIGHT(A2, LEN(A2)-FIND("分隔符",A2))
  • 提取两个分隔符之间： =MID(A2, FIND("分隔符1",A2)+1, FIND("分隔符2",A2)-FIND("分隔符1",A2)-1)
  • 这三个公式模板可以解决大多数变长字段的提取需求， 记住模板比记住每个参数的含义更实用。
