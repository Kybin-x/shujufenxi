3.1 认识数据与数据分类处理

识别数据问题类型，掌握基本处理思路
📁 配套文件：3.1_练习数据.xlsx 🎯 三个难度等级 🏢 情景：星途电商有限公司

📋 背景情景

星途电商有限公司是一家主营 3C 配件（手机数据线、充电宝、手机壳、蓝牙耳机等）的电商企业。 运营部小李负责整理近期平台导出的订单数据，发现数据中存在各种质量问题，需要你协助完成数据问题的识别与分类工作， 为后续数据清洗做准备。

✅ 任务要求
  1. 打开 CSV 文件，将数据复制到 3.1_练习数据.xlsx 的"数据（基础）"Sheet 中。
  2. 逐行检查每条数据，判断该行是否存在问题。
  3. 在 Excel 中新增一列"问题类型"，在对应行填写问题类型名称（缺失值 / 格式错误 / 异常值 / 逻辑错误），无问题则填"正常"。
  4. 在 Excel 中再新增一列"问题说明"，用一句话描述具体问题，例如："销售额为空"、"日期格式为斜线分隔"。
  5. 完成后统计：四种问题各发现了多少条？填入下方记录表。

💡 操作提示
  • 可以使用 Excel 的"条件格式"→"突出显示空值"快速定位缺失值。
  • 日期格式问题：观察单元格内容，格式不是 yyyy-mm-dd 的均属格式错误。
  • 异常值判断参考：数据线/充电宝/手机壳单价一般在 10～299 元之间，超出此范围需标记。
  • 逻辑错误：重点检查"订单状态"与"销售额"是否匹配（已完成但金额为0属逻辑错误）。
  • 新增列可在 F 列（问题类型）和 G 列（问题说明）操作，不要修改原始数据。

✅ 任务要求
  1. 打开数据文件，将数据复制到"数据（巩固）"Sheet 中。
  2. 逐行识别问题：注意一行数据可能同时存在多种问题，需全部列出。
  3. 在 Excel 中新增"问题类型（全部）"列，用"/"分隔同一行的多个问题，例如："缺失值/格式错误"。
  4. 新增"处理方法"列，说明针对每种问题你建议采用哪种方法（补齐 / 删除 / 格式转换 / 人工核实）。
  5. 新增"预计处理结果"列，简述处理后该字段/行会变成什么样。
  6. 完成后填写下方汇总表。

💡 操作提示
  • 多问题并存的识别方法：建议逐列检查，先检查所有行的某一列，再换下一列，避免遗漏。
  • 处理方法判断依据：
    ① 推广花费为空 → 可用0填充（说明无推广费用）
    ② 日期格式混乱 → 格式统一转换
    ③ 销售额为负但状态为"已完成" → 人工核实
    ④ 发货状态缺失且销售额为0 → 可删除整行
  • 注意"预计处理结果"要具体，例如："将2024/3/15统一改为2024-03-15"。
LEVEL 3

进阶练习

适合：综合运用阶段 · 35 条数据
📊 数据文件：进阶数据（35条）｜字段数：9列｜含隐蔽逻辑错误
📌 数据字段说明
字段名 说明 示例
订单编号 平台订单号 ORD-2024-201
商品类别 商品大类 数据线 / 充电宝 / 手机壳 / 蓝牙耳机
SKU编码 商品唯一规格码 DXX-BK-1M
销售额（元） 成交金额 89.00
退款金额（元） 退款金额，无退款为0 0 / 89.00
退款率 退款金额÷销售额 0% / 100%
下单时间 精确到分钟 2024-03-15 10:23
签收时间 买家签收时间 2024-03-18 14:05
客服备注 客服人工填写的说明 客户投诉 / 正常 / （空）
✅ 任务要求
  1. 完整检查 35 条数据，识别所有问题（包括表面问题和隐蔽的逻辑问题）。
  2. 在 Excel 中建立"诊断表"（可另起一个 Sheet），包含以下列：行号 / 问题类型 / 问题字段 / 问题描述 / 严重程度（高/中/低） / 处理建议。
  3. 重点关注以下隐蔽逻辑错误（需结合多列判断）：
    • 签收时间 早于 下单时间（时间逻辑矛盾）
    • 退款率 ≠ 退款金额÷销售额（计算逻辑矛盾）
    • 退款率异常高（某商品类别整体退款率超过50%）
    • 销售额为0但订单状态不是退款/取消
  4. 完成后撰写一份简短的数据诊断报告（在 Excel 中新建"诊断报告"Sheet，用文字说明）：
    • 发现了哪些问题（分类列举）
    • 哪些问题优先处理（说明理由）
    • 针对每类问题给出具体处理建议
💡 操作提示
  • 时间逻辑判断：在新增列中用公式 =IF(签收时间<下单时间,"⚠️时间异常","正常") 快速标记。
  • 退款率验证：新增辅助列 =退款金额/销售额，与"退款率"列对比，不一致即为逻辑错误。
  • 严重程度参考标准：
    高：影响业务决策的关键字段错误（如销售额、时间逻辑）
    中：影响分析准确性的格式/计算问题
    低：不影响分析的格式不统一、备注缺失等
  • 诊断报告不要求长篇大论，每个要点用 2～3 句话说清楚即可。
