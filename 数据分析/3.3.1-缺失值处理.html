3.3.1 缺失值处理

识别、判断并处理数据中的空值，保障分析结果的准确性
📁 配套文件：3.3.1_练习数据.xlsx 🎯 三个难度等级 🏢 情景：星途电商有限公司

📋 背景情景

星途电商有限公司仓储物流部门负责管理供应链数据， 包括商品入库、出库、库存预警等记录。 由于数据由多人分批录入，常常出现字段遗漏的情况。 数据分析实习生小周被安排处理这批"问题数据"， 需要对每个字段的缺失情况做出判断，并选择合适的处理方式。 请协助小周完成缺失值的识别与处理工作。

📖 知识点速览：缺失值的三种处理方式

✅ 补齐填充
适用场景：
  • 该字段对分析很重要，不能删除
  • 可以通过其他字段推算出来
  • 用固定值（如0、均值）填充有业务意义
例：推广花费为空 → 填0（无推广）
🗑️ 删除处理
适用场景：
  • 该行数据缺失太多，保留无意义
  • 关键字段（如订单编号）为空
  • 缺失比例极低，删除影响不大
例：订单编号为空且销售额为空 → 删整行
⏸️ 保留不处理
适用场景：
  • 该字段对本次分析不重要
  • 缺失本身有业务含义（如备注为空=无异常）
  • 等待业务方补充后再处理
例：客服备注为空 → 保留（无异常无需备注）
⚠️ 快速定位缺失值：【开始】→【查找和选择】→【定位条件】→ 勾选"空值"→ 确定， Excel 会自动选中所有空单元格。也可使用条件格式高亮显示空值。

✅ 任务要求
  1. 将数据粘贴到"数据（基础）"Sheet，使用"定位条件→空值"功能，快速定位所有空单元格并用黄色高亮标记。
  2. 逐列检查，在新增的"处理方式"列（H列）中，为每个有缺失的行填写处理决定： 补齐 删除整行 保留不处理
  3. 在新增的"处理说明"列（I列）中，用一句话说明做出该决定的理由。
  4. 按照决定执行处理操作：
    • 需要补齐的：直接在空单元格填入合适的值
    • 需要删除的：右键行号→删除整行
    • 保留不处理的：不做任何操作
  5. 完成后填写下方"处理记录汇总表"。

💡 操作提示
  • 快速定位空值：按 Ctrl+G → 点击"定位条件" → 勾选"空值" → 确定， 所有空单元格会被选中，然后设置填充色（黄色）即可高亮显示。
  • 商品类别缺失：可以根据"商品名称"字段推断类别（如"快充数据线"属于"数据线"类），属于可补齐情况。
  • 入库数量缺失：数量字段是关键业务数据，如果无法推断，建议标记为"待确认"而非随意填0。
  • 质检备注缺失：备注字段缺失通常意味着"无异常"，可保留不处理，或统一填写"无"。
  • 单价缺失：可查找同名商品的其他行单价进行补齐（同类商品单价应一致）。

✅ 任务要求
  1. 将数据粘贴到"数据（巩固）"Sheet，统计各字段的缺失数量，填入字段统计表。
  2. 制定缺失值处理策略：先不做操作，在"策略表"区域（可在 Sheet 右侧空白区建立） 写出每个字段的处理决定，格式如下：
    补齐→ 说明补什么值； 删除整行→ 说明什么情况下删； 保留→ 说明为何保留
  3. 对于同时缺失多个字段的行，需综合判断：若缺失字段 ≥ 3 个关键字段（渠道/展现量/点击量/花费/销售额），建议删除整行。
  4. 按策略逐步执行处理操作，每完成一步，在操作记录中打钩。
  5. 完成后填写处理结果汇总，并回答分析问题。

💡 操作提示
  • 统计缺失数量：可用 =COUNTBLANK(B2:B31) 函数统计某列的空单元格数量。
  • 策略优先于操作：处理缺失值前一定要先想清楚策略，不要边看边改，容易遗漏或误操作。
  • 自然搜索的花费：如果某行推广渠道明确是"自然搜索"，但花费字段为空，可以填0（自然搜索无推广费用）。
  • 重度缺失行的判断：没有绝对标准，只要理由充分即可。通常缺失超过50%字段的行建议删除。
  • 操作前建议先复制一份原始数据到新Sheet备份，防止误操作后无法还原。
LEVEL 3

进阶练习

适合：综合运用阶段 · 35条供应链数据 · 含业务逻辑推算，撰写处理报告
📊 数据说明：35条供应链数据｜9个字段｜部分缺失值可从其他字段推算补全
📌 数据字段说明
字段名列位置说明字段关联关系
批次编号A列入库批次唯一号独立字段
商品名称B列商品全称可推断商品类别
商品类别C列充电宝/数据线/手机壳/蓝牙耳机/磁吸充电器可由商品名称推断
入库数量（件）D列本批次入库数量入库金额÷单价可推算
单价（元）E列商品采购单价入库金额÷入库数量可推算
入库金额（元）F列= 入库数量 × 单价数量×单价可推算
供应商G列供货方名称同名商品供应商通常相同
入库日期H列实际入库时间独立字段，不可推算
质检结果I列合格/不合格/待检独立字段，不可推算
🔢 关键字段的推算逻辑
缺失字段 推算公式 前提条件 Excel公式示例
入库金额 = 入库数量 × 单价 入库数量和单价均不为空 =D2*E2
单价 = 入库金额 ÷ 入库数量 入库金额和入库数量均不为空 =F2/D2
入库数量 = 入库金额 ÷ 单价 入库金额和单价均不为空 =F2/E2
商品类别 根据商品名称人工判断 商品名称不为空 手动填写
供应商 查找同名商品的其他行 同名商品有已知供应商 =VLOOKUP(B2,查找区域,列号,0)
✅ 任务要求
  1. 将数据粘贴到"数据（进阶）"Sheet，用"定位条件→空值"全面统计缺失情况。
  2. 第一步：推算补全。 对可以通过公式推算的缺失字段，用公式补全：
    • 入库金额 = 入库数量 × 单价（当两者都有时）
    • 单价 = 入库金额 ÷ 入库数量（当两者都有时）
    • 商品类别缺失：根据商品名称人工判断填写
  3. 第二步：无法推算的缺失值处理。 对无法推算的字段（如入库日期、质检结果缺失）， 根据业务判断选择补齐、删除或保留，并说明理由。
  4. 第三步：撰写处理报告。 在新 Sheet（命名"处理报告"）中，按下方模板撰写完整的缺失值处理报告。
🌳 缺失值处理决策流程（逐步判断）
  1. 能否从其他字段推算出来？
    → 能推算 → 用公式补齐
    → 不能推算 → 进入下一步
  2. 该字段对本次分析是否关键？
    → 关键字段 → 进入下一步
    → 非关键字段（如备注）→ 保留不处理
  3. 该行其他字段是否完整？
    → 其他字段完整，只有1-2个缺失 → 人工补齐或标记"待确认"
    → 该行多处缺失（≥3个关键字段）→ 删除整行

📄 处理报告模板（在"处理报告"Sheet 中完成）
报告结构内容要求
① 数据基本情况 总条数、字段数、处理前总缺失数、缺失率最高的字段
② 推算补全情况 共推算补全了哪些字段、多少处、用了什么公式，推算结果是否合理
③ 删除行情况 共删除了几行、删除依据是什么（哪些字段缺失），对数据完整性的影响
④ 保留未处理的缺失 哪些缺失选择保留，原因是什么
⑤ 处理后数据质量评估 处理后共剩余多少条数据、多少处缺失，是否满足后续分析需要
⑥ 改进建议 针对数据录入环节，提出1-2条减少缺失值的建议
💡 操作提示
  • 推算前先验证逻辑：找几行三个字段（数量、单价、金额）都完整的数据，验证"数量×单价=金额"是否成立，确认逻辑正确后再用公式补全缺失行。
  • 供应商推算：可用 =VLOOKUP(B2, $B$2:$G$36, 6, 0) 查找同名商品对应的供应商（第6列），但要确认同名商品是否确实只有一个供应商。
  • 入库日期缺失：日期字段无法推算，若缺失比例不高（<10%），可标注"日期未知"保留；若某行整体数据可疑，可删除。
  • 质检结果缺失：可联系业务方补充，在 Excel 中标注"待补充"，不要随意填"合格"。
  • 处理报告字数不需太多，每项要点用2-3句话说清楚，重点是数据和理由要对应。
