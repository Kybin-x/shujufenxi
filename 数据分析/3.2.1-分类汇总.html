3.2.1 分类汇总

按指定字段对数据进行分组统计，快速生成多层级汇总报告
📁 配套文件：3.2.1_练习数据.xlsx 🎯 三个难度等级 🏢 情景：星途电商有限公司

📋 背景情景

星途电商有限公司运营部助理小王需要向主管汇报本月各商品类别的销售情况。 主管要求按商品类别和销售省份分层呈现数据， 小王决定使用 Excel 的分类汇总功能快速完成统计。 请你根据不同难度的要求，协助小王完成数据汇总任务。

📖 知识点速览：分类汇总的核心要点

  1. 使用前必须先排序：必须先按"分类字段"排序，否则同一类别会被拆分成多个汇总行。
  2. 汇总方式可选择：求和、计数、平均值、最大值、最小值等。
  3. 嵌套汇总：先按主字段汇总，再按次字段汇总（第二次汇总时取消勾选"替换当前分类汇总"）。
  4. 取消汇总：数据→分类汇总→全部删除，可恢复原始数据状态。
⚠️ 重要提醒：分类汇总功能位于【数据】选项卡→【分级显示】组→【分类汇总】。 进行嵌套汇总时，第二次操作不能勾选"替换当前分类汇总"。

✅ 任务要求
  1. 打开 CSV 数据，粘贴到 3.2.1_练习数据.xlsx 的"数据（基础）"Sheet 中。
  2. 确认数据已按"商品类别"列升序排列（数据已预先排好，直接观察确认即可）。
  3. 使用"分类汇总"功能，按商品类别对销售额进行求和汇总。
  4. 展开/折叠汇总结果，查看各级别视图（点击左侧数字 1、2、3 切换）。
  5. 将汇总结果填写到下方"结果记录表"中。
  6. 完成后在 Excel 中写出操作步骤（可在"任务书"Sheet 中用文字描述）。
🔧 操作路径提示
  1. 点击数据区域任意单元格
  2. 点击菜单栏【数据】选项卡
  3. 找到【分级显示】组，点击【分类汇总】
  4. 在弹出对话框中设置：
    · 分类字段 → 商品类别
    · 汇总方式 → 求和
    · 选定汇总项 → 勾选销售额（元）
  5. 点击【确定】，查看汇总结果

💡 操作提示
  • 本数据已预先按"商品类别"排好序，可以直接进行分类汇总，无需手动排序。
  • 汇总完成后，左侧会出现 1 2 3 三个级别按钮：
    · 点击 1 = 只显示总计行
    · 点击 2 = 显示各类别小计+总计
    · 点击 3 = 显示所有明细数据
  • "占总销售额比例"需要自己手动计算（用类别合计÷总计），可在记录表旁边列用公式计算。
  • 如果操作错误想重来：【数据】→【分类汇总】→【全部删除】。

✅ 任务要求
  1. 打开 CSV 数据，粘贴到"数据（巩固）"Sheet 中，不要修改原始数据，在旁边空白区域或新 Sheet 中操作。
  2. 进行两列排序：先按商品类别升序，再按销售省份升序 （使用"高级排序"，添加两个排序条件）。
  3. 完成第一层汇总：按商品类别对销售额求和。
  4. 完成第二层汇总：再次打开分类汇总对话框，按销售省份对销售额求和， 取消勾选"替换当前分类汇总"。
  5. 切换到第 2 级视图，将各省各类别的销售额汇总结果填写到记录表中。
  6. 根据汇总结果，回答分析问题。
🔧 操作路径提示
  1. 排序步骤：【数据】→【排序】→ 添加条件：主要关键字"商品类别"升序，次要关键字"销售省份"升序 → 确定
  2. 第一层汇总：【数据】→【分类汇总】→ 分类字段"商品类别"，汇总方式"求和"，汇总项"销售额" → 确定
  3. 第二层汇总：再次点击【分类汇总】→ 分类字段"销售省份"，汇总方式"求和"，汇总项"销售额"→ 取消勾选"替换当前分类汇总" → 确定

💡 操作提示
  • 嵌套汇总的关键：第二次操作时必须取消勾选"替换当前分类汇总"，否则第一层汇总会被覆盖。
  • 排序顺序很重要：主要关键字是"商品类别"，次要关键字是"销售省份"，顺序不能反。
  • 如果省份种类较多，记录表中"其他省份"可合并填写汇总数。
  • 切换视图：左侧出现 1234 四个级别按钮，点击 3 可查看省份级别汇总。
