3.4.3 DATEDIF 日期间隔计算

计算会员复购间隔、物流时效、客户活跃度，实现客户分层管理
📁 配套文件：3.4.3_练习数据.xlsx 🎯 三个难度等级 🏢 情景：飞跃跨境电商有限公司

📋 背景情景

飞跃跨境电商有限公司客服部和运营部需要从时间维度分析业务数据： ① 会员多久购买一次（复购间隔）？② 订单从下单到签收用了多少天（物流时效）？ ③ 某位客户有多长时间没来购买了（流失风险）？ 这些都需要计算日期之间的间隔，Excel 中最适合的工具就是 DATEDIF 函数。

📖 知识点速览：DATEDIF 函数

=DATEDIF( 开始日期, 结束日期, "单位" )

计算两个日期之间的间隔，结果为整数。注意：开始日期必须早于结束日期，否则报错。

"D"
间隔天数（最常用）
=DATEDIF(A2,B2,"D") → 30
"M"
间隔完整月数
=DATEDIF(A2,B2,"M") → 2
"Y"
间隔完整年数
=DATEDIF(A2,B2,"Y") → 1
"MD"
不足一月的天数（余天）
忽略月份差
"YM"
不足一年的月数（余月）
忽略年份差
TODAY() 函数：返回今天的日期，配合 DATEDIF 可计算"距今"多少天：
=DATEDIF(A2, TODAY(), "D") → 计算从 A2 的日期到今天的天数
⚠️ 注意：DATEDIF 不会出现在 Excel 的函数自动补全列表中，但可以正常使用，直接输入即可。 开始日期必须≤结束日期，否则返回 #NUM! 错误，请用 IFERROR 保护。

✅ 任务要求

  2. 在 G2 输入公式计算复购间隔月数，向下填充：=DATEDIF(C2, D2, "M")

  4. 统计三个频级各有多少人，填入统计表。
  5. 回答分析问题。

= IF( G2<=3, "高频", IF( G2<=_____ , "中频", "低频" ) )
第二个IF的条件：G2≤6（即6个月以内为中频）

💡 操作提示
  • DATEDIF 计算月数（"M"）说明：结果是完整的月数，不足1个月的部分截去。例如"2023-06-01"到"2024-02-15"之间是8个完整月。
  • IF嵌套的逻辑顺序：先判断最严格的条件（≤3），再判断次严格的（≤6），最后是剩余情况，这样逻辑最清晰。
  • COUNTIF统计频级：=COUNTIF(H2:H26,"高频")
  • 验证：三个频级人数之和应等于25。
LEVEL 2

巩固练习

适合：熟悉基本函数后 · 28条订单数据 · 计算发货时效+签收时效+是否超时
📊 数据说明：28条订单物流数据｜含下单/发货/签收三个时间点｜ 计算发货时效+签收时效+是否超时，统计超时率
📌 数据字段说明
字段名列说明特殊情况
订单编号A唯一订单号—
商品类别B商品大类—
下单日期C客户下单时间—
发货日期D商家发货时间部分未发货，为空
签收日期E买家签收时间部分在途/未收，为空
发货时效（天）F【待计算】下单→发货天数发货日为空时显示"未发货"
签收时效（天）G【待计算】发货→签收天数签收日为空时显示"配送中"
全程时效（天）H【待计算】下单→签收总天数未签收时显示"待完成"
发货是否超时I【待计算】发货时效>2天为超时—
✅ 任务要求
  1. 将数据粘贴到"数据（巩固）"Sheet，在 F、G、H、I 列建立计算列。
  2. 发货时效（F列）：用 IFERROR 包裹 DATEDIF，发货日为空时显示"未发货"：
    =IFERROR(DATEDIF(C2,D2,"D"), "未发货")
  3. 签收时效（G列）：发货→签收，签收日为空时显示"配送中"：
    =IFERROR(DATEDIF(D2,E2,"D"), "配送中")
  4. 全程时效（H列）：下单→签收，未签收时显示"待完成"：
    =IFERROR(DATEDIF(C2,E2,"D"), "待完成")
  5. 发货是否超时（I列）：发货时效>2天为"超时"，≤2天为"正常"，未发货为"未发货"：
    =IFERROR(IF(F2>2,"超时","正常"), "未发货")
  6. 统计超时率，填写分析表，并分类别分析超时情况。
✏️ 关键公式填写
1发货时效（F列）：下单→发货天数
D列（发货日期）为空时，DATEDIF报错，IFERROR捕获显示"未发货"
= IFERROR( DATEDIF( C2, D2, "_____" ), "未发货" )
单位用"D"表示天数
2签收时效（G列）：发货→签收天数
= IFERROR( DATEDIF( D2, _____ , "D" ), "配送中" )
结束日期为 E2（签收日期）；E列为空时显示"配送中"
3发货是否超时（I列）：IFERROR + IF嵌套
F列为"未发货"（文本）时，F2>2的比较会报错，IFERROR捕获显示"未发货"
= IFERROR( IF( F2 > _____ , "超时", "正常" ), "未发货" )
超时阈值为2天（发货时效>2天即超时）

💡 操作提示
  • IFERROR捕获顺序：DATEDIF在开始日期或结束日期为空时会报错，IFERROR统一捕获并显示自定义文字。
  • I列"超时"判断的报错原因：F列中有"未发货"（文本），当用 IF(F2>2) 比较文本时Excel会报错，所以需要IFERROR在外层捕获。
  • AVERAGEIF跳过文本：=AVERAGEIF(F2:F29,"<未发货") 会忽略文本值，只对数值行求平均。但更精确的方法是：=AVERAGEIF(I2:I29,"<>未发货",F2:F29)
  • 超时率 = 超时订单数 ÷ 已发货订单数（不含"未发货"行）。

✅ 任务要求
  1. 将数据粘贴到"数据（进阶）"Sheet，在 H-L 列建立五个计算列。
  2. 会员时长（H列）：首次购买日期到今天的完整月数：
    =DATEDIF(C2, TODAY(), "M")
  3. 距上次购买天数（I列）：最近购买日期到今天的天数：
    =DATEDIF(D2, TODAY(), "D")
  4. 平均购买周期（J列）：会员时长内平均多少天买一次：
    =IFERROR(H2*30/E2, "--")（用月数×30近似换算成天数，再除以购买次数）
  5. 活跃度分级（K列）：根据I列天数，用IF嵌套分4级。
  6. 流失风险标记（L列）： 综合判断——距上次购买超过"2倍平均购买周期"的客户标记为"⚠️高风险"， 否则标记为"正常"：
    =IFERROR(IF(I2 > J2*2, "⚠️高风险", "正常"), "正常")
  7. 完成下方客户分层分析报告。

1会员时长（H列）：TODAY() 动态计算

= DATEDIF( C2, TODAY(), "_____" )
单位用"M"（月份）
2距上次购买天数（I列）
= DATEDIF( D2, _____ , "D" )

3活跃度分级（K列）：四级IF嵌套
≤30天→活跃，≤90天→关注，≤180天→沉睡，其余→流失
= IF( I2<=30, "活跃", IF( I2<=90, "关注", IF( I2<=_____ , "沉睡", "流失" ) ) )
第三层IF的阈值为180天

= IFERROR( IF( I2 > J2 * _____ , "⚠️高风险", "正常" ), "正常" )
阈值为2（即超过2倍购买周期）

💡 操作提示
  • TODAY() 的特性：每次打开 Excel 文件，TODAY() 都会自动刷新为当天日期，因此计算结果是动态的。这意味着本练习中的"距上次购买天数"每天都在增加。
  • 平均购买周期的近似计算：=H2*30/E2 用月数×30近似换算为总天数，再除以购买次数，得到平均多少天买一次。这是近似值，精确算法需要用到首末购买日期之差。
  • COUNTIFS 统计"活跃+高风险"组合：=COUNTIFS(K:K,"活跃",L:L,"⚠️高风险")
  • 筛选高风险客户：在 L 列上使用筛选器，只显示"⚠️高风险"，然后复制可见行到报告表。
  • 召回价值评估参考：累计消费额越高、购买次数越多的高风险客户，价值越大，优先投入召回资源。
