💰
基础销售指标
销售额
单价 × 销售数量
💡 卖了多少钱
利润
销售额 - 成本
💡 赚了多少钱
利润率
利润 ÷ 销售额 × 100%
💡 每卖100元赚多少
客单价
总销售额 ÷ 买家人数
💡 平均每个顾客花了多少钱
案例：坚果大礼包（单价89.9元，销售50件，成本2800元）
// 销售额
=B2*C2 → 89.9 × 50 = 4,495 元
=D2-E2 // 利润 → 4495 - 2800 = 1,695 元
=F2/D2 // 利润率 → 1695÷4495 ≈ 37.7%
💡计算出利润率后，选中单元格 → 右键 →「设置单元格格式」→「百分比」，设置小数位数即可显示为百分比格式。

推广效果指标
ROI
成交金额 ÷ 推广花费
💡 花1元赚几元
PPC 点击单价
推广花费 ÷ 点击次数
💡 每次点击花多少钱
点击率 CTR
点击次数 ÷ 展现次数
💡 展示100次点几次
案例：直通车-爆款（展现10万次，点击3千次，花费1500元，成交12000元）
=E2/D2 // ROI → 12000÷1500 = 8（每花1元带来8元销售额）
=D2/C2 // PPC → 1500÷3000 = 0.5元（每次点击花0.5元）
=C2/B2 // CTR → 3000÷100000 = 3%（每100次展示有3次点击）
💡ROI怎么看？ ROI > 1 说明推广赚钱；ROI < 1 说明推广亏钱。ROI = 8 表示投入1元赚回了8元，效果非常好。具体多少算"好"，要结合行业平均水平和商品利润率来判断。

DATEDIF 函数——计算日期间隔
函数格式
=DATEDIF(起始日期, 结束日期, "单位代码")
通俗理解：从某天到某天，一共过了多久？
代码含义举例（2024/1/15 到 2024/8/20）
"Y"相差的完整年数0（不满1年）
"M"相差的完整月数7个月 ← 最常用
"D"相差的总天数218天 ← 最常用
"YM"不足一年的月份差7个月
"MD"不足一个月的天数差5天
⚠️结束日期必须晚于起始日期！如果写反了（结束日期比起始日期早），函数会返回 #NUM! 错误。确保 date1（起始日期）在前、date2（结束日期）在后。
案例：计算会员客户复购间隔
客户编号会员名首次下单(C列)最近下单(D列)间隔月数 E列公式结果
QL001张先生2022/3/182024/10/1=DATEDIF(C2,D2,"M")30个月
QL002李小姐2023/1/192024/5/11=DATEDIF(C3,D3,"M")15个月
QL003王同学2024/1/32024/7/12=DATEDIF(C4,D4,"M")6个月 ← 复购最快
QL004赵先生2023/6/112024/8/12=DATEDIF(C5,D5,"M")14个月
💡运营分析：张先生30个月才复购，属于低频客户，可以通过优惠券或会员活动来提高复购频率；王同学仅6个月就复购，是需要重点维护的忠实客户。
⏰
TODAY() 函数 + 日期直接相减
三种常用日期计算方式
=TODAY() // 自动获取今天的日期，每天打开文件自动更新
=D2-C2 // 日期直接相减，结果为天数（最简单）
=TODAY()-C2 // 从某日期到今天过了多少天
=IF(D2>15,"需催促","未超时") // 配合IF函数判断是否超时
案例：标记评价超时的订单
订单会员名确认收货(C列)距今天数(D列)是否超时(E列)
D001张先生2024/10/7=TODAY()-C2=IF(D2>15,"需催促","未超时")
D002李小姐2024/10/1=TODAY()-C3=IF(D3>15,"需催促","未超时")
💡物流时效计算直接用日期相减即可：=签收日期-发货日期，结果就是物流天数。这是最简单的日期计算方式。
DAYS360 函数（了解即可）
格式与用途
=DAYS360(起始日期, 结束日期)
按每月固定30天计算（一年360天），常用于金融领域的账期计算。日常电商运营中，直接用日期相减或 DATEDIF 更常用。

什么是加权计算？
🎓
生活中的加权：期末成绩
老师说：平时成绩占30%，期末考试占70%。你的平时成绩80分，期末90分：

简单平均：(80 + 90) ÷ 2 = 85分
加权计算：80 × 30% + 90 × 70% = 24 + 63 = 87分

因为期末权重更大，期末考好了会拉高总成绩。这就是加权——每个数据不是平等对待，而是乘以各自的权重（重要程度）后再求和。
📌加权计算在电商中的应用：综合评估推广渠道效果（ROI权重最大）、商品综合评分（销量、好评率、利润率各有权重）、客户价值评分等。所有权重加起来必须等于100%。
⚡
SUMPRODUCT 函数——加权计算的利器
格式与原理
=SUMPRODUCT(数组1, 数组2)
原理：将两组数据"对应相乘"，再把所有乘积"加起来"。通俗理解：先一对一地乘，再全部加起来。
SUMPRODUCT 计算过程可视化（渠道A的综合得分）
得分 8×10% + 3×20% + 3×40% + 3×30%
结果 0.8 + 0.6 + 1.2 + 0.9 = 3.5 分

绝对引用（$ 符号）的作用

在公式 =SUMPRODUCT(B2:E2,I$2:L$2) 中，I$2:L$2 使用了 $ 符号，叫做绝对引用。

引用方式向下拖动填充柄时的变化效果
相对引用 I2:L2会变成 I3:L3、I4:L4……引用到了权重行下面的其他行❌ 计算错误
绝对引用 I$2:L$2始终保持为 I$2:L$2，权重行始终正确✅ 计算正确
💡记忆口诀：$ 符号放在行号前面 = "锁住行号不让它跑"。I$2 意思是：I列可以变，但第2行被锁住。

📌 本单元核心要点
1
数据计算分三大类型：常规计算、日期计算、加权计算，各有核心工具
2
建议用单元格引用写公式（=B2*C2 而非 =89.9*50），数据变化时结果自动更新
3
电商9大核心指标：销售额=单价×数量、转化率=下单数÷访客数、ROI=成交额÷推广费、客单价=总额÷买家数等，务必记住
4
DATEDIF(起始日期, 结束日期, 代码)：代码"Y"=年、"M"=月、"D"=天，结束日期必须晚于起始日期
5
TODAY() 自动获取当天日期，每天打开文件自动更新，适合做"距今多少天"的动态计算
6
SUMPRODUCT(数组1, 数组2)：将两组数据对应相乘再全部加起来，是加权计算的核心函数
7
$ 符号（绝对引用）锁住行号或列号，向下拖动填充柄时保持引用不变——权重区域必须用 I$2:L$2 而非 I2:L2
