// PART_02 · KNOWLEDGE

六步流程 · 逐一精讲

从目标确定到报告撰写，掌握数据分析的完整方法论

⏱ ~22 min
📌 核心知识
电子商务数据分析的六步流程
1
明确数据分析目标 展开 ▼
根据分析目标选择需要分析的数据，明确想要达到什么效果。先决条件，为后续的数据采集、处理、分析提供清晰的指引方向。
明确目标包括三个层次：
① 分析目的：为什么要做这次分析？（例：找出销售下滑原因）
② 分析框架：从哪几个角度分析？用哪些指标？（例：流量、转化率、客单价）
③ 分析方法：针对每个角度确定具体方法和指标

✅ 好的目标

"分析本月成交转化率下降的原因，并提出优化建议"——具体、可执行、有方向

❌ 差的目标

"分析一下我们的数据"——太模糊，无法指引分析方向

2
数据采集 展开 ▼
数据来源分为直接获取（内部系统/统计调查/科学实验）和间接获取（查阅资料/数据统计工具）两种方式。

🏪 直接获取

通过统计调查或科学实验获取第一手数据
• 店铺后台数据（淘宝/天猫生意参谋）
• 问卷调查
• 客服记录

🌐 间接获取

通过查阅资料、数据统计工具获取
• 行业报告（艾瑞咨询等）
• 竞品数据平台
• 政府统计数据

3
数据处理 展开 ▼
对采集到的数据进行加工整理，目的是从杂乱数据中提取有价值的信息，是数据分析前必不可少的阶段。

🧹 数据清洗

删除重复信息、纠正错误、保证数据一致性（例：删除重复订单、修正错误日期格式）

🔄 数据转化

将原始数据转换为适合分析的形式（例：将销售额从元转换为万元，统一单位）

📤 数据提取

从数据源中抽取特定数据（例：只提取某类目的销售数据进行分析）

🔢 数据计算

加减乘除运算，最大化开发数据价值（例：计算各品类的毛利率和增长率）

4
数据分析 展开 ▼
用适当的分析方法及工具，对处理过的数据进行分析，提取有价值信息，形成有效结论的过程。

// PART_03 · CASE

实战演练：为「潮趣女装店」做一次完整数据分析

按照6步流程，逐步完成一次真实场景的数据分析任务

⏱ ~15 min
// MISSION BRIEF

任务：本季度转化率下滑，找出根本原因

潮趣女装店 Q3 数据显示，成交转化率从上季度的 3.2% 下降到 2.1%，老板急需知道原因。
你作为数据分析师，需要按照 6 步流程，从"明确目标"到"形成报告"完成这次分析。
// STEP 01
明确目标
目标：分析 Q3 成交转化率下滑原因，找出主要影响因素
分析框架搭建指标拆解
// STEP 02
数据采集
来源：生意参谋后台（直接），行业报告（间接）
生意参谋行业均值
// STEP 03
数据处理
清洗重复数据，统一日期格式，计算各渠道转化率
数据清洗数据计算
04
数据分析：对比分析法定位问题
// 点击展开分析过程
▼
分析思路：销售额 = UV × 成交转化率 × 客单价，三个维度逐一排查。

// 对比分析结果
UV本期: 18,500   UV上期: 16,200   ↑ 14.2%（无问题）
客单价本期: ¥268   客单价上期: ¥271   ≈ 持平（无问题）
转化率本期: 2.1%   转化率上期: 3.2%   ↓ 34.4%（❗️关键问题）
// 进一步拆解转化率
PC端转化率: 1.8%   移动端转化率: 2.3%   移动端跌幅更大
主图点击率: 4.1%   行业均值: 5.8%   ↓ 显著低于行业
结论：转化率下滑主要由移动端主图点击率偏低引起，访客量充足但进入详情页的比例偏低。
05
数据展现：选择合适图表呈现结论
// 图表选择策略
▼
图表方案：

📈 折线图：展示 Q1~Q3 成交转化率走势，直观看出 Q3 骤降
📊 柱状图：对比本期 vs 上期 vs 行业均值的主图点击率
🔻 漏斗图：展示"访客 → 进入详情页 → 加购 → 付款"各环节转化情况，直接定位卡点在哪

漏斗图分析结果：
访客 18,500 → 点击详情页 3,700（20%）→ 加购 1,480（40%）→ 付款 389（2.1%）
卡点在第一步：从访客到点击详情页仅 20%，行业均值为 35%，差距明显
06
撰写报告：结论与建议
// 最终交付物
▼
// 分析结论
Q3 成交转化率下降 34.4%，根本原因在于：
1. 移动端主图点击率 4.1%，低于行业均值 5.8%，差距 29%
2. 访客进入详情页比例仅 20%，行业均值 35%
// 运营建议
1. 主图优化：A/B 测试 3 套移动端主图，重点突出产品卖点
2. 标题优化：加入核心关键词，提升搜索匹配度
3. 详情页：首屏加载速度优化，3s内完成加载
4. 复盘周期：每周一统计上周数据，及时调整
✅ 这份报告符合"结构清晰、图文并茂、科学严谨"的三个标准，可以直接呈给老板参考。
