🤔 什么是趋势分析？先想想这个场景

🚫 这不叫趋势分析
老板看了看销量折线图，说："哦，这个月比上个月多了一些。"
→ 只是"看图"，没有分析原因，没有预测未来
✅ 这才是趋势分析
数据分析师说："由于双十一活动，11月销量激增，预测12月将达到XXX万元，建议提前备货。"
→ 分析原因 + 预测未来 + 给出建议

趋势分析 = 在已有数据的基础上，利用科学方法，对未来市场需求、发展方向做出判断，为营销决策服务。核心是：明确数据变化 + 分析变化原因（内部/外部）

🎯 怎么判断趋势线准不准？——R²（R平方值）

R²越接近1，说明趋势线越能"解释"数据，预测越准确

0
不准一般较准很准
1
< 0.5
❌ 拟合较差，参考价值低
0.5~0.8
⚠️ 一般，可用但要注意
> 0.8
✅ 优秀！预测可信度高
💡 通俗理解
比如气象预报：R²=0.9，说明天气预报有90%能被这个模型解释，很准！ R²=0.4，说明模型只能解释40%，预测不太靠谱，需要换模型。

📉 六种趋势线，如何选择？

看数据"长什么样"决定选哪种
📏
线性趋势线
🎯 数据近乎直线增长/下降
增长或降低速度比较平稳，数据点基本构成直线形态。
✔ 举例：企业年销售额平稳增长
📈
指数趋势线
🎯 增速越来越快（曲线上扬）
增长幅度越来越大，曲线越来越陡。
⚠️ 注意：数据中有零或负值时不能用！
✔ 举例：爆款商品刚上架的销量增长
📉
对数趋势线
🎯 先快后慢，逐渐趋于平缓
增长一开始快，后来越来越慢，最终趋于平稳。
✔ 举例：新店初期涨粉快，后来越来越慢
〰️
多项式趋势线
🎯 数据有明显波动（有峰有谷）
数据波动大，有高峰和低谷。阶数=波动次数（2阶=1个波峰）。
✔ 举例：电商节日促销—普通日的销售波动
⚡
乘幂趋势线
🎯 增速稳定持续增加
增长速度持续增加，且增加幅度比较恒定。
⚠️ 注意：数据含零或负数时不能用！
✔ 举例：平台用户数量的稳定增长
〜
移动平均趋势线
🎯 消除随机波动，看清总体趋势
取特定期数的平均值，平滑处理数据中的小波动。
✔ 举例：每日销量忽高忽低，想看整体走向

🛠️ 三个实战操作案例

📦 案例背景
某店铺2010-2018年历年销售额数据（单位：万元），需要预测2019年和2020年的销售额。数据变化较平稳，适合用线性趋势线。
1

制作折线图

打开Excel文件，选择数据区域（A2:B12），插入带数据标记的折线图

选择 A2:B12 → 插入 → 图表 → 折线图 → 带数据标记的折线图
2

添加线性趋势线

选中图表 → 图表工具 → 设计 → 添加图表元素 → 趋势线 → 线性

选中图表 → 设计 → 添加图表元素 → 趋势线 → 线性(L)
3

设置格式并获取公式

双击趋势线 → 设置趋势线格式 → 向前预测2年 → 勾选"显示公式"

Excel自动生成的回归公式：
y = 187.6x + 334.78
x = 数据点序号，y = 对应年份销售额
4

计算预测值

2019年是第10个数据点(x=10)，2020年是第11个数据点(x=11)

📅 2019年预测
y = 187.6×10 + 334.78
≈ 2210.78 万元
📅 2020年预测
y = 187.6×11 + 334.78
≈ 2398.38 万元
🔢 自己试试！用线性公式计算预测值
公式：y = 187.6 × x + 334.78（x = 数据点序号）
万元（预测销售额）
📦 案例背景
某店铺近10个月销量数据（单位：万件），销量呈加速增长态势，适合用指数趋势线。需预测11月、12月销量。
1

制作散点图

选择A2:B13 → 插入 → 散点图 → 仅带数据标记的散点图

选择 A2:B13 → 插入 → 散点图 → 仅带数据标记
2

添加指数趋势线

选中图表 → 设计 → 添加图表元素 → 趋势线 → 指数

3

获取指数公式与R²值

双击趋势线 → 向前预测2期 → 勾选"显示公式"和"显示R平方值"

y = 23.661e^(0.1385x)
R² = 0.9279 ✅ 拟合很好！
4

计算11月、12月预测销量

📅 11月（第11个数据点）
y = 23.661×e^(0.1385×11)
≈ 108.56 万件
📅 12月（第12个数据点）
y = 23.661×e^(0.1385×12)
≈ 124.69 万件
📦 案例背景
分析某店铺8月份销售额与销售费用的关系，数据呈U形波动，适合用二阶多项式趋势线，并预测销售额为300万时的销售费用。
最终获得的预测公式（二阶多项式）：
y = 0.0014x² - 0.6466x + 85.109
R² = 0.64（中等，需要注意预测误差）
x= 销售额（万元）
y= 销售费用（万元）
✍️ 应用计算

已知8月21日销售额为300万元，代入公式：

y = 0.0014×300² - 0.6466×300 + 85.109
= 0.0014×90000 - 193.98 + 85.109
= 126 - 193.98 + 85.109
≈ 17.13 万元

⏰ 什么是时间序列？

📋 时间序列的定义

按时间顺序排列的一组数据，时间间隔相等。

按年排列 按季度 按月 按天

🔬 基本原理

① 承认延续性：历史趋势会延伸到未来

② 消除随机性：用统计处理消除偶然波动

优缺点对比

✅ 优点

📊 分析过去、现在、未来联系效果好
🔢 数据处理不复杂，操作相对简单
📈 对短期稳定趋势预测准确度高

⚠️ 缺点

📉 只适合预测稳定延续的过程
❌ 不适合长期预测（误差会累积）
🔮 不考虑因果关系，可能遗漏重要因素

🛠️ 三种常用预测方法

🌊

季节波动法

按季节规律预测，考虑淡旺季影响

📊

移动平均法

用最近几期平均值平滑预测下期

⚖️

指数平滑法

近期数据权重更大，适应趋势变化

🌊 季节波动法是什么？
由于季节变化（天气、节假日、消费习惯等）导致销量有规律地忽高忽低。
例如：羽绒服冬天卖得好，防晒霜夏天卖得好。
季节波动法就是利用这种规律性来预测未来各季度的销量。
操作步骤（共5步）：
1

收集至少3年以上的各季度历史数据

数据要完整，如有极端值（如突发疫情影响的季度）需删除

2

计算各季度的同季平均值（用A表示）

把同一季度5年的数据加起来求平均

=AVERAGE(B3:B7) → 计算第一季度5年平均值
3

计算所有季度的总平均值（用B表示）

=AVERAGE(B8:E8) → 计算四季度综合平均值
4

计算季节指数 S = A ÷ B

季节指数>1：该季度高于平均水平（旺季）；<1：低于平均（淡季）

=B8/B9 → 第一季度季节比率
5

计算预测值 = 全年预测均值 × 季节指数

=F11/4*B10 → 2019年第一季度预测值
📊 移动平均法是什么？
用"最近几期"的平均数来预测下一期。每次都取相同数量的数据求平均，"移动"就是随时间推进，去掉最早的，加上最新的。
适合：数据既不快速增长也不快速下降，且无明显季节性的场景。
Excel操作步骤：
1

用公式计算一次移动平均

选择D8单元格，输入公式计算前12个月的平均值，然后向下填充

=AVERAGE(C3:C14) → 向下填充至D20
2

计算二次移动平均（平滑曲线）

=AVERAGE(D8:D9) → 向下填充至E20
3

或使用"数据分析"工具（更方便！）

数据 → 数据分析 → 移动平均 → 确定 输入区域：利润列数据 间隔(N)：12（取12期移动平均） ☑ 图表输出 ☑ 标准误差
💡 标准误差越小 = 预测越准确！
⚖️ 指数平滑法是什么？
给"近期数据"更大权重，"远期数据"权重小。通过平滑系数α来控制权重分配。
注意：α（平滑系数）= 1 - 阻尼系数，Excel中设置的是阻尼系数。
核心公式：
下期预测数 = 本期实际数 × α + 本期预测数 × (1-α)
α = 平滑系数（0~1之间）；α越大，近期数据权重越大
🔢 指数平滑法在线计算器
（下期预测值）
Excel操作步骤：
数据 → 数据分析 → 指数平滑 输入区域：B3:B13（生产量数据列） 阻尼系数：0.6（即平滑系数α=0.4） 输出区域：C3 ☑ 图表输出 → 确定
💡 结果解读：系统自动生成预测图，12月预测值 = 1122×0.4 + 1105.62×0.6 ≈ 1112.17万件
