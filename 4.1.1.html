💡 什么时候用哪个？

📊 用平均值，当：
• 数据比较均匀，没有极端值
• 想了解整体平均水平
• 如：班级平均成绩
📊 用中位数，当：
• 数据有极端值（特别大或特别小）
• 想了解"大多数人"的水平
• 如：收入水平、房价分析
📊 用众数，当：
• 想知道"最受欢迎"是什么
• 如：卖得最多的尺码、最热门的价格段

三大类统计量详解

选择类别查看详情

💬 集中趋势 = 数据集中在哪个水平？就是"大家的整体表现怎样"

📊
Mean · 平均值

平均值

所有数据相加除以个数。最常用，但容易被极端数据影响。

举例：20天访客量平均 = 全部加起来 ÷ 20

🎯
Median · 中位数

中位数

排好顺序后，最中间那个数。不受极端值影响，更真实。

举例：20个数排序，取第10、11个的平均值

🏆
Mode · 众数

众数

出现次数最多的数。反映"最普遍"的情况。

举例：6860访客量出现了2次，是众数

➕
Sum · 和

总和

所有数据之和。如计算总销售额、总访客量等。

举例：20天总访客量 = 126904人次

💬 离散程度 = 数据波动有多大？差异有多悬殊？越大说明越不稳定

📏
Range · 极差

极差（全距）

最大值 - 最小值。反映数据的总体跨度，越大说明波动范围越广。

案例：10923-3127=7796（访客量跨度很大！）

📉
Std.deviation · 标准差

标准差

描述数据偏离平均值的程度。标准差越大，数据越分散、越不稳定。

案例：标准差2231.38，说明每天访客量波动较大

📐
Variance · 方差

方差

标准差的平方。和标准差一样用于衡量离散程度，但单位不同。

记住：方差 = 标准差² ，实际中更常用标准差

⬆️
Maximum / Minimum

最大值 / 最小值

数据的最高点和最低点。帮助发现异常情况。

案例：最大值10923（双十一），最小值3127（活动初期）

🎯
S.E.mean · 均值标准误差

均值标准误差

反映抽样误差的大小，越小说明平均值越可靠、越准确。

案例：498.95，表示平均值估计的可靠性

💬 分布情况 = 数据是左多右少，还是均匀分布？用偏度和峰度来描述

📐 偏度（Skewness）——数据是否对称？
👈
偏度 < 0
左偏：大多数数据集中在右边，少数极小值拉向左
⚖️
偏度 ≈ 0
对称：左右两边数据分布均匀，是"正态分布"
👉
偏度 > 0
右偏：大多数数据集中在左边，少数极大值拉向右
📈 峰度（Kurtosis）——数据集中还是分散？
🏔️
峰度 > 0
尖峰：数据高度集中，极端值少
🔔
峰度 ≈ 0
正态：标准钟形曲线，最理想状态
🏝️
峰度 < 0
平峰：数据较分散，分布比较均匀
💡 记忆口诀：偏度峰度都接近0，就是正态分布！实际工作中，偏度在±1之间、峰度在±2之间，基本可认为接近正态分布。

📍 集中趋势

中位数 Median排序后中间值

极差 Range最大-最小

方差 Variance标准差²
最大/最小值数据边界

📉 分布情况

偏度 Skewness≈0为对称
峰度 Kurtosis≈0为正态
两者都≈0→ 正态分布✔
S.E.mean越小越可靠

🛠️ Excel操作步骤（跟着做！）

1

添加"数据分析"加载项

如果你的 Excel 数据菜单右上角没有"数据分析"按钮，需要先开启它：

文件 → 选项 → 加载项 → 管理(Excel加载项) → 转到 勾选【分析工具库】和【分析工具库-VBA】 → 点击确定 ✔ → 数据菜单右上角出现"数据分析"按钮 🎉
2

打开描述统计工具

打开"活动访问数据.xlsx"文件，启动分析工具：

点击顶部【数据】选项卡 → 点击【数据分析】按钮 → 在列表中找到并选择【描述统计】 → 点击【确定】
3

设置参数（重要！）

在弹出的"描述统计"对话框中，按照下面的参数表进行设置：

☑ 勾选
第一行是"UV"标题，需要勾选

$D$2

✍️ 分析结论怎么写？

从描述性统计分析结果可以看出：在双十一活动期间（11月1日-20日），该店铺日均访客量为 6345人次。 随着平台各种推广活动的开展，访客量在双十一前持续增长，并于 11月11日达到最高峰（10923人次）， 是活动初期最低值（3127人次）的 3.5倍。标准差为2231，说明活动期间各天访客量波动较大。 活动结束后访客量有所回落，但整体高于活动初期， 说明本次双十一活动对提升店铺流量有显著效果，品牌曝光价值持续显现。
